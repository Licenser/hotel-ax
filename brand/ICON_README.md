# Brand Icon

You need to create an `icon.png` file in this directory for HACS compatibility.

## Requirements

- **Size**: 256x256 pixels (or 128x128)
- **Format**: PNG
- **Name**: `icon.png`

## Suggestions

Since this is **Hotel-AX** (a play on The Shining's "Overlook Hotel"), consider:

- A stylized axe icon
- Home Assistant logo + Axiom logo composite
- The Overlook Hotel facade (public domain imagery)
- Abstract geometric design with "AX" or hotel theme

You can use tools like:
- [Canva](https://canva.com)
- [DALL-E](https://openai.com/dall-e)
- [Figma](https://figma.com)

Or grab a public domain/CC0 icon from:
- [The Noun Project](https://thenounproject.com)
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)

Once created, place it here as `brand/icon.png`.
